# Mi primer bot de telegram con Telebot

Este repositorio contiene el codigo para crear un bot basico de telegram utilizando la biblioteca pyTelegramBotAPI de Python.

## Configuracion

1. Clona este repositorio
2. Instala las dependencias utilizando `pip install -r requirements.txt`
3. Crea un bot en telegram a través de BotFather y obtén tu token
4. Reemplaza 'TU_TOKEN_AQUI' en 'main.py' con tu token
5. Ejecuta el bot `main.py`

## Funcionalidades

- Responde a los comandos `\start` y `\help`
- Responde a los textos: `entrenamiento` y `clima`
- Repite cualquier otro mensaje que le envíen
